## 1. Overview

The Kestrel 4500 Pocket Weather Tracker is Kestrel's environmental meter. This is a mini weather station in itself :
- Heading
- Wind direction
- Crosswind
- Headwind
- Altitude
- Barometric pressure
- Wet bulb temperature
- Relative humidity in %
- Density altitude
- Wind chill
- Temperature °C
- Current, average, and maximum air velocity
- Time and date
- User screen 1
- User screen 2
